# m-biyisport-archive

## 项目简介

`m-biyisport-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅作为静态 HTML 文件的长期存储与展示空间。  
所有页面均为独立构建，内容来源广泛，旨在保留网页快照或离线可访问的版本。

## 目录说明

```
.
├── pages/            # 存放所有归档的 HTML 页面文件
├── assets/           # 页面引用的公共资源（CSS、JS、图片等）
├── index.html        # 仓库首页，提供页面索引或导航
└── README.md         # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的归档页面，命名方式为 `page-xxx.html` 或按主题归类。
- `assets/`：存放页面所需的样式、脚本、字体或图片，避免外部资源失效。
- `index.html`：可选的主页，用于列出所有归档页面并附上简要说明。

## 页面归档说明

- 所有归档页面均为完整的 HTML 文件，包含独立的样式与脚本，可直接在浏览器中打开。
- 归档时尽可能保留原始页面结构、内容与基本样式。部分交互功能可能因外部依赖失效而简化。
- 页面内不包含任何跟踪代码、动态广告或外部请求（除非必须保留以维持原始体验）。
- 归档时间与来源信息会记录在页面头部注释或 `index.html` 的对应条目中。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/m-biyisport-archive.git
   ```
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件即可浏览。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
